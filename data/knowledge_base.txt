## 公司介绍
DP 本店"极客数码港"成立于 2020 年，主营智能手机、笔记本电脑、平板电脑、智能穿戴设备及数码配件。所有商品均为正品行货，享受全国联保。公司总部位于深圳市南山区科技园，在北京、上海、广州、成都设有仓储中心。

## 退货政策
DP 本店支持 7 天无理由退货（自签收之日起计算）。退货商品需保持原包装完好、配件齐全、不影响二次销售。以下商品不支持无理由退货：(1) 已激活的手机/平板/电脑等智能设备；(2) 已拆封的耳机类商品；(3) 虚拟商品及软件序列号；(4) 定制类商品。
DP 退货时请附上购买凭证（订单号或收据）。退货邮费：非质量问题由买家承担寄回运费；质量问题由本店承担来回运费（上限 20 元，超出部分需协商）。
DP 退款时效：收到退货商品并核验通过后，款项将在 3-5 个工作日内原路退回。如超过 7 个工作日未收到退款，请立即联系客服核查。
DP 退货流程：APP/网站 → 我的订单 → 申请退货 → 填写原因 → 等待审核（1-2小时内）→ 审核通过后寄回商品 → 仓库签收核验 → 退款到账。

## 换货政策
DP 收到商品后 15 天内可申请换货（自签收之日起计算）。换货仅限同款同色同规格商品。如需更换其他型号，请走退货流程后重新下单。
DP 因商品质量问题的换货：本店承担来回运费，换货处理周期为 3-7 个工作日。
DP 因个人原因的换货（尺码不对、颜色不喜欢等）：运费由买家承担，换货周期为 5-10 个工作日。
DP 换货商品需保持原包装完好、配件齐全。如原包装已丢弃或损坏，可能影响换货处理。
DP 换货流程：APP/网站 → 我的订单 → 申请换货 → 填写原因 → 客服审核（2小时内）→ 寄回商品 → 仓库签收 → 发出新商品。

DP 下单后 48 小时内安排发货（预售商品以页面标注的发货时间为准）。每日 16:00 前下单当天发货，16:00 后次日发货。

DP 丢件处理：如物流信息超过 7 天未更新且无法确认包裹位置，可申请为丢件。核实属实后，本店将在 24 小时内为您补发新商品或全额退款。

## 手机/平板类 FAQ
Q: 手机激活后还能退吗？
A: DP 根据《网络购买商品七日无理由退货暂行办法》，已激活的手机/平板/电脑等智能设备，如无质量问题不支持无理由退货。因为激活后会产生数据痕迹，影响二次销售。请在激活前仔细检查外观和基础功能。

Q: 国外买的手机能在这里保修吗？
A: DP 本店销售的是中国大陆行货版本，享受全国联保。非大陆版本（港版/美版/日版等）不在本店销售范围内，也不享受大陆官方保修。

Q: 手机屏幕碎了保修吗？
A: DP 屏幕碎裂属于人为损坏，不在标准保修范围内。建议购买本店的"碎屏险"服务（购机时可加购，年费 99 元起），可享受免费换屏一次。

Q: 电池不耐用了怎么办？
A: DP 手机/平板电池属于消耗品，正常使用下电池健康度在一年内低于 80%，可享受免费更换。超出一年或健康度在 80% 以上，更换电池需收费（价格因型号而异，一般在 150-500 元之间）。

## 电脑类 FAQ
Q: 电脑可以加装内存/硬盘吗？
A: DP 部分型号支持自行升级，具体请查看商品详情页的"可扩展性"说明。自行拆机升级将不影响正常保修（前提是升级过程未造成其他损坏）。如需本店协助升级，可在下单时选择"定制升级"服务。

Q: 预装系统是什么？
A: DP 笔记本电脑预装 Windows 11 家庭中文版（已激活）或 macOS（苹果设备）。部分高性能机型预装无系统版（DOS/Ubuntu），价格更低，适合自行安装系统的用户。

Q: 散热不好怎么办？
A: DP 新机在运行大型软件或游戏时发热属正常现象。建议将电脑放置在硬质平面上使用，避免在床/沙发等软面上使用阻碍散热。如日常使用（浏览网页/看视频）也出现明显发烫，可能是散热系统异常，请及时联系客服。

## 穿戴设备类 FAQ
Q: 手表/手环防水吗？
A: DP 不同型号防水等级不同，请以商品详情页标注为准。一般生活防水(IP67)可防溅水/汗液/小雨；游泳级防水(5ATM/IP68)可在游泳/浅水活动时佩戴。但热水淋浴/桑拿/深潜均不建议佩戴任何电子设备。

Q: 表带可以更换吗？
A: DP 大部分智能手表的表带支持快拆更换，本店提供硅胶/皮质/金属/尼龙等多种材质的替换表带。部分型号使用专用接口，需购买对应品牌的专用表带。

## 耳机/音频类 FAQ
Q: 耳机有底噪/电流声？
A: DP 蓝牙耳机在安静环境下有轻微底噪属正常现象（蓝牙芯片工作声）。如底噪明显到干扰正常听音，可能是品控问题，请联系客服处理。

Q: 降噪效果不理想？
A: DP 主动降噪对低频持续噪音（如引擎声/空调声）效果显著，对人声/高频突发声音效果有限。请确保耳塞尺寸合适、佩戴到位。如仍感降噪效果异常，可联系客服送检。
